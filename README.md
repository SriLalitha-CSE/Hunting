
# Color Palette Generator

A modern, interactive web app that generates beautiful color palettes with one click.
Built using **HTML, CSS, and JavaScript**.

---

##  Features

* 🎨 Generate random color palettes
* 🔒 Lock colors you like
* 📋 Copy hex codes instantly
* 🌈 View RGB values
* ⌨️ Keyboard shortcuts support
* 🌙 Dark mode toggle
* 💾 Save palettes in local storage
* 🖼️ Click-to-copy colors
* ⚡ Smooth UI interactions

---

## Keyboard Shortcuts

| Key     | Action               |
| ------- | -------------------- |
| `Space` | Generate new palette |
| `L`     | Lock all colors      |
| `U`     | Unlock all colors    |
| `C`     | Copy full palette    |

##  How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Click **Generate** or press `Space`
4. Start creating palettes 🎨

---

##  How It Works

* JavaScript generates random hex colors using `Math.random()`
* Each color column updates dynamically
* Locked colors stay unchanged during regeneration
* Clipboard API is used for copying colors
* `localStorage` saves user palettes
